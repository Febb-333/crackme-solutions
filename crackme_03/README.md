# Solusi Crackme 03: Dekonstruksi Algoritma Generator Kunci / Keygen (Medium Level)

- **Sumber Tantangan:** crackmes.one
- **Kakas yang Digunakan:** Ghidra Decompiler & Python 3
- **Target Operasi:** Membongkar rumus logika matematika kalkulasi lisensi serial.

### Langkah Analisis:
1. Hasil dekompilasi pseudokode bahasa C di Ghidra menunjukkan bahwa program tidak menyimpan string password secara statis.
2. Program menerapkan fungsi pengecekan algoritma dinamis terhadap input pengguna.
3. Aturan algoritma lisensi yang tertera di dalam kode:
   - Panjang karakter input lisensi wajib berjumlah tepat 5 karakter.
   - Karakter indeks pertama (paling depan) harus merupakan huruf kapital 'A'.
   - Empat karakter sisanya wajib berupa kombinasi angka numerik bebas.

### Solusi Pembuatan Keygen:
Berdasarkan visualisasi logika matematika di atas, saya merancang sebuah skrip otomatisasi menggunakan bahasa pemrograman Python untuk memproduksi serial lisensi valid dalam jumlah massal secara otomatis.
